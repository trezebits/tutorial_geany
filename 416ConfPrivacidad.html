    Privacidad¶

    Material for MkDocs hace que el cumplimiento de las regulaciones de privacidad de datos sea muy fácil, ya que ofrece una solución nativa de consentimiento de cookies para buscar el consentimiento explícito de los usuarios antes de configurar análisis. Además, los activos externos se pueden descargar automáticamente para su autoalojamiento.

    Material for MkDocs facilita el cumplimiento de las normas de privacidad de datos, integrando una solución de consentimiento de cookies.

    Consentimiento de cookies¶

    Se muestra un consentimiento de cookies configurable. Añadir a mkdocs.yml algo así como:

    extra:
  consent:
    title: Consentimiento de cookies
    description: >- 
      Este sitio web utiliza cookies para reconocer sus visitas repetidas y sus preferencias, 
      así como para medir la efectividad de nuestra documentación y si los usuarios
      encuentran lo que están buscando. Con su consentimiento, nos está ayudando a
      mejorar nuestra documentación.

    Podemos añadir el texto HTML que deseemos, por ejemplo, un enlace a una página de términos del servicio.

    Las propiedades en el apartado consent pueden ser:

    • title - Muestra un título en el formulario de consentimiento de cookies
    • description - Texto que se muestra bajo el título. Puede incluir etiquetas HTML, por ejemplo, un enlace a los términos del servicio.

    • cookies - Permite añadir cookies personalizadas o cambiar el estado inicial y nombre de las cookies incluidas con el tema. En la actualidad son:

      • Google Analytics. Muestra: analytics (activada por defecto)
      • GitHub. Muestra: github (activada por defecto)

      Cada cookie debe recibir un identificador único que se utiliza como clave en el mapa de cookies y se puede configurar en una cadena o en un mapa que define el nombre y el estado marcado:

      extra:
  consent:
    cookies:
      analytics: 
        name: Nombre personalizado
        checked: false

      Si Google Analytics ha sido configurado via mkdocs.yml, el consentimiento de cookie incluirá una marca para que el usuario la desactive. Podemos crear cookies personalizadas utilizando JavaScript.

    • actions [accept, manage]. Esta propiedad define que botones mostrar y en que oreden, por ejemplo, aceptar las cookies o gestionarlas:

      extra: consent: actions: - accept - manage

      El consentimiento de cookies incluye tres tipos de botones:

      • accept
      • reject
      • manage

    El formulario se muestra cuando se visita el sitio por primera vez.

    cambiar consentimientos¶

    Para cumplir con la normativa, el usuario debe poder cambiar su configuración de cookies en cualquier momento. Esto puede hacerse con un enlace en el copyright:

    copyright: >
  Copyright &copy; 2016 - 2024 Mi nombre –
  <a href="#__consent">Cambiar consentimiento de cookies</a>

    Elementos externos¶

    El complemento de privacidad incorporado identifica automáticamente los activos externos como parte del proceso de construcción y descarga todos los activos para un autohospedaje simple. Agregue las siguientes líneas a mkdocs.yml:

    plugins:
  - privacy

    Cookies personalizadas¶

    Si personalizó el consentimiento de cookies y agregó una cookie personalizada, se le pedirá al usuario que acepte o rechace su cookie personalizada. Una vez que el usuario acepta o rechaza el consentimiento de las cookies, o cambia la configuración, la página se recarga. Utilice JavaScript adicional para consultar el resultado:

    extra_javascript:
  - javascripts/consent.js

    Y en consent.js:

    var consent = __md_get("__consent")

if (consent && consent.custom) {
  /* The user accepted the cookie */

} else {
  /* The user rejected the cookie */
}
